This repository contains code that allows you to gather twitter feeds for movies and then analyses them using a naive-bayes classifier.

Description for files- 

1. app.py - This file gathers twitter feeds for movies specified in the file named 		    movies and then creates a file with twitter feeds , their geo location , 		    the retweet count, user's screen name and the date on which it was posted

	    For running this code you require a configuration file named conf.

2. conf	  - This configuration file is a normal text file which points to the required 
	    file
